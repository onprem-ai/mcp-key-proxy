# Contributing

## Getting started

```bash
git clone https://github.com/digilac/mcp-key-proxy.git
cd mcp-key-proxy
npm install
npm test
```

## Pull requests

- Open PRs against the `main` branch.
- Use [Conventional Commits](https://www.conventionalcommits.org/) for commit messages: `feat`, `fix`, `docs`, `chore`.
- Run `npm run typecheck && npm test` before submitting.
- Keep `CHANGELOG.md` updated under the `[Unreleased]` section.
